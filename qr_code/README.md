# QR Codes

Pour se connecter plus facilement, vous pouvez générer des qr codes de connexion aux point d'accès et aux pages web avec les scripts python `qr_code_ap.py` et `qr_code_moderation.py`.

## Dépendances

Installation des dépendances python :

``` shell
pip3 install -r requirements.txt
```

## Générer les qr codes de connexion

Créer les qr codes de connexion :

``` shell
python3 qr_code_ap.py
```

Créer les qr codes d'accès aux pages web :

``` shell
python3 qr_code_moderation.py
```
